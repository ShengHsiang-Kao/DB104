聯邦這張信用卡，我把它放在第三位 ( 常用信用卡 )！

Pi 錢包，一般消費是 2.5 % P 幣回饋，保費就交給這張信用卡了。超商 ( 全家、OK 、萊爾富 ) 則為 4.5 % 及 3.5 % 回饋。Pi 錢包目前在這部份仍比聯邦強！ ( 基本上，超商部份，Pi 錢包，我只會使用全家 4.5 %回饋 ，OK 、萊爾富，我會使用 台新 A-GoGo  搭配 街口支付，這樣就有 3.5 % 現金回饋 )

華南 I 網購信用卡，首年回饋是 5 % 現金回饋。網路訂購、支付工具 ( 例如：街口支付 … 等 ) , 就交給這張信用卡

第三張則是聯邦這張。起碼聯邦這張要比中國信託 Line Pay 信用卡要強多了 。

聯邦這張若有搭到 Line Pay 活動，( line points 加碼回饋 )，則有時會瞬間成為第一位信用卡。

總之，實體信用卡的話，必須超過 2.5 % , 數位 ( 包括網購 ) 則必須 4 % 以上回饋，否則替代性就會很快被替代了。

期待 西元 2019年下半年能再出現「信用卡神卡」 ( 黑馬爆出 )！

聯邦這張，若沒有搭配 Line pay 活動的話，只能說稍有及格邊緣，但很有可能會被後面的黑馬所取代。希望聯邦這張能再多釋出回饋空間，才能拉長它的龍頭之位。
